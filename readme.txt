Embed any song to your wordpress site by just typing the name of the song into a shortcode.
 
== Description ==

**SongLark API Player** allows you to place any song in the world on your site just by using a simple shortcode. Pretty cool huh?

= USAGE: = 
**Embed a song:**

	[embed_songlark query="Tongue Tied by Grouplove"]


== Installation ==
I recommend installing and activating this plugin from the WordPress admin panel under **Plugins > Add New**. If you must download and install separately:

1. Upload the **SongLark-API-Player** folder to **/wp-content/plugins/**
1. Activate the plugin through the **Plugins** menu in WordPress.
